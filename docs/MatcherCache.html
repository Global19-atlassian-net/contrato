MatcherCache

new MatcherCache()

Create a matcher cache object

Methods

add(matcher, value)

Add a value to the matcher cache

Parameters:
Name Type Description
matcher Object matcher contract
value Any cached value
Example
const cache = new MatcherCache()
const matcher = Contract.createMatcher({
  type: 'sw.os',
  slug: 'debian'
})

cache.add(matcher, {
  foo: 'bar'
})
